# Aquawel

Сайт-витрина мебельного производства Aquawel на Next.js.

## Запуск

```bash
npm install
npm run dev
```

Главная страница доступна на `http://localhost:3000`.
Каталог коллекций доступен на `http://localhost:3000/catalog`.

## Каталог

Каталог реализован как витрина коллекций: пользователь выбирает коллекцию через верхний переключатель, после чего на странице отображается набор изделий этой коллекции. В карточке изделия есть изображение, название, цена, краткое описание, список параметров и кнопка заявки.

Данные коллекций сейчас отдаются через placeholder API:

```env
NEXT_PUBLIC_COLLECTIONS_API_URL=/api/catalog
NEXT_PUBLIC_LEAD_API_URL=/api/leads
```

Для подключения реального backend достаточно заменить значения переменных в `.env`.
